# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-01

### Added

- Firebase integration for help requests management
- Contacts directory with filtering and Firebase integration
- Admin dashboard and moderation UI with Firestore smoke test
- Lebanese flag as app favicon and header logo
- Google reCAPTCHA v3 to protect public forms
- Open Graph image and meta tags for link previews
- Dynamic OG image for social sharing thumbnails
- Smart matching, demo banner, tests, and UX improvements
- Interactive Lebanon heatmap and browse page filter improvements
- Anti-fraud features: moderation queue, verification, delivery confirmation
- Approved shelter centers page with daily-cached ArcGIS data
- Interactive map to shelters page
- Example `.env` file for Firebase configuration
- Shelter data source disclaimer
- Bilingual SVG letter badges on shelters page
- GitHub Actions CI workflow (lint, type check, test, build)

### Fixed

- Logo and title link to home, increased map size on mobile
- Filter chips and toggle sizes for better mobile tap targets
- Governorate URL param type casting for build
